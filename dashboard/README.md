# Dashboard

The Power BI dashboard (`upi_pulse.pbix`) is not tracked in this repository
as it requires a local PostgreSQL connection to function.

## To run the dashboard
1. Set up the project following the main README instructions
2. Run `scripts/generate_data.py` to populate the database
3. Open `upi_pulse.pbix` in Power BI Desktop
4. Connect to your local PostgreSQL instance when prompted
